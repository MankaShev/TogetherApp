TogetherApp — это прототип Android-приложения для создания и управления подборками мест на карте.

Основная идея приложения — позволить пользователям сохранять интересные места и объединять их в коллекции.

# Используемые технологии

В проекте используются следующие технологии:

- Kotlin
- Android SDK
- XML Layout
- MVVM архитектура
- Yandex MapKit API
- Git и GitHub
## Основной сценарий

Пользователь может:

1. Открыть приложение
2. Войти в приложение
3. Перейти на главный экран
4. Создать новую подборку
5. Добавить место с карты
6. Сохранить подборку
7. Увидеть её в списке

## Архитектура

Проект реализован с использованием **MVVM** и элементов **Clean Architecture**.

Структура слоев:

UI (Activity)  
↓  
ViewModel  
↓  
UseCase  
↓  
Repository
